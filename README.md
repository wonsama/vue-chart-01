# README.md

> vue 기반으로 차트를 보여주는 것을 목표로 함. ( vite, typescript, vue )

## 설치 및 실행

degit 로 프로젝트 복제 이후 수정 작업

```sh
degit https://github.com/wonsama/vue-chart-01.git vue-chart-01
cd vue-chart-01
yarn
yarn dev
```

> [로컬 미리보기](http://localhost:3000/)
