# pre_01_install_with_vite

> 공식 홈에서도 vite 사용을 추천 한다고 해서 `vite` 로 설치 완료

```sh
yarn create vite
baby vue-chart-01 (project name)
over-write
vue (framework)
vue-ts (variant)

cd vue-chart-01
yarn
yarn dev
```
